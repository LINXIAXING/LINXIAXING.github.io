环境搭建

在做调用前，需要先将训好的模型导出为ONNX。

C#下的模型调用需要添加 Microsoft.ML Microsoft.ML.OnnxRuntime Microsoft.ML.OnnxRuntime.Managed System.Drawing.Common 四个NuGet包。但是在 Unity 中，只能给 C# 脚本引用 dll 或者安装 NuGet 包，而不能给 C# 项目做 dll 引用。

所以借助第三方工具 NuGetForUnity 实现NuGet包安装管理：Github Path

安装方式与unity导包一样，不赘述。System.Drawing.Common用于图像处理，语言模型用不着

安装后在工具栏将多出NuGet选项，进入包管理，查询三个包并安装。
